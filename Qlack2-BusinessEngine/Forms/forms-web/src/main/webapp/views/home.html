Welcome to the Forms component

The Forms module combines all the functionalities necessary to define and use custom XForms in a business application. As with other QBE components users can use the Forms Development Environment to create and edit available forms. The Forms Development Environment provides a text editor for editing the XForms source, while a forms editor is also available for editing forms graphically. This editor is provided by Orbeon, which is used to provide XForms functionalities to the Forms Runtime, and therefore support all functionalities supported by the Orbeon Form Builder. The Forms Development environment can therefore be used by QBE users in order to:
  • Create and edit forms.
  • Create form templates which can be used as building blocks of other forms.
  • Manage form versions.
  • Organise forms using categories and tags.
QBE also supports defining a set of business rules which should be used during form validation in order to provide more advanced forms processing.
