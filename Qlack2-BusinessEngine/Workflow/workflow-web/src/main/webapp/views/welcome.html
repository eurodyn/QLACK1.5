Welcome to the Workflow component

The Workflow module provides the necessary functionality for QBE users to create and manage workflows which can later be leveraged by their business applications. The Workflow Development Environment which is included in the QBE UI allows users to perform all actions related to the creation and management of workflows, such as the following:
  • Upload workflow definitions in BPMN2.0 format to create new workflows.
  • Create and manage different versions for each workflow.
  • Organise workflows with categories and tags to allow for easier reference.
Business applications can then use the QBE API to spawn new instances of specific workflows and perform tasks such as advancing a workflow to the next steps or retrieving and executing human tasks related to a running workflow.
